Zoopla Crawler Settings

Backend

Set your API base URL (no trailing /api/properties). Example: http://localhost:8000.

The backend must expose POST /api/properties and accept JSON: { \"properties\": [...] }.

Khi không dùng Backend

Nếu để trống Backend URL, dữ liệu lưu local. Đạt đủ số bản ghi bên dưới thì extension tự export CSV và tải xuống máy, rồi xóa bản ghi local để tiếp tục đợt mới.
